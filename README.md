# OkHttp-Demo
一个简易的通过OkHttp框架实现的网络通信交互demo
#### 客户端：Android
#### 服务端：JAVA
做了一个登陆操作的Demo，服务端接收json数据，通过Gson解析数据后，返回客户端（android）结果，
客户端采用Okhttp框架，发送http请求，并得到服务器返回的结果，解析。
